# PocketMovie
영화 박스오피스 API, 영화상세정보 검색 API를 활용한 영화 정보 검색 어플
### 화면 구조
- 메인화면 (박스오피스 순위 화면)
- 검색화면
- 영화 상세정보 화면
- 영화 웹사이트 화면
